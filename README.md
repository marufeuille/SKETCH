# スケッチアプリ

シンプルで使いやすいブラウザベースのスケッチアプリです。自由に描画したり、写真を追加したりして、アイデアやメモを視覚的に表現できます。

## 機能

- **描画機能**

  - ペンツール：自由に線を描画
  - 消しゴムツール：描画内容を消去
  - 色選択：6 色のカラーパレットから選択可能
  - サイズ選択：3 種類のサイズから選択可能
  - クリア機能：キャンバス全体をクリア

- **写真管理機能**

  - 写真のアップロード：ローカルファイルから写真を追加
  - 写真の配置：ドラッグ＆ドロップで自由に配置
  - 写真の削除：不要な写真を削除

- **保存機能**
  - 作品を PNG 形式で保存
  - 日付を含むファイル名で保存

## ファイル構成

```
/
├── css/
│   └── sketch.css      # スタイルシート
├── js/

│   ├── drawing.js      # 描画機能
│   ├── photo.js        # 写真管理機能
│   └── save.js         # 保存機能
├── sketch.html         # メインのHTMLファイル
├── package.json        # プロジェクト設定
└── README.md           # このファイル
```

## 使用方法

1. HTTP サーバーを起動します：

   ```
   python -m http.server
   ```

2. ブラウザで以下の URL にアクセスします：

   ```
   http://localhost:8000/sketch.html
   ```

3. スケッチアプリの使い方：
   - ツールバーからペンまたは消しゴムを選択
   - 色とサイズを選択
   - キャンバス上で描画
   - 「写真を追加」ボタンで写真をアップロード
   - 「選択」ツールで写真を選択し、ドラッグして配置
   - 「保存」ボタンでスケッチを画像として保存

## 開発者向け情報

### コードの構造

- **名前空間**
  - `SketchApp`：アプリケーション全体の名前空間
  - `SketchApp.Drawing`：描画機能
  - `SketchApp.Photo`：写真管理機能
  - `SketchApp.Save`：保存機能

### 拡張方法

新機能を追加する場合は、以下のファイルを修正します：

- 描画ツールの追加：`js/drawing.js`
- 写真関連機能の拡張：`js/photo.js`
- 保存形式の変更：`js/save.js`
- スタイルの変更：`css/sketch.css`
